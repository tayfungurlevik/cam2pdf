cam2pdf - Akıllı Belge Tarayıcı ve OCR
cam2pdf, Android cihazınızı güçlü bir taşınabilir tarayıcıya dönüştüren modern bir uygulamadır. Google'ın en yeni ML Kit teknolojilerini kullanarak belgeleri tarar, metinleri tanır (OCR) ve kartvizitlerdeki bilgileri otomatik olarak ayıklayabilir.
Android Version Kotlin License
🚀 Özellikler
•
Belge Tarayıcı: ML Kit Document Scanner API ile kağıt belgeleri yüksek kalitede dijital PDF'e dönüştürün.
•
Metin Tanıma (OCR): Resimlerdeki metinleri otomatik olarak tanıyın ve kopyalanabilir metne çevirin.
•
Varlık Ayıklama (Entity Extraction): Kartvizitler üzerindeki telefon numarası, adres ve e-posta gibi bilgileri otomatik olarak tespit eder.
